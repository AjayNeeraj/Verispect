## The founder's daily routine (the discipline that hits the number)

**07:30 — Solo standup (5 min).** Read yesterday's tracker row. Say out loud: yesterday's closes, today's #1 priority, biggest blocker. Write today's close target.

**08:00–11:00 — OUTBOUND (sacred block, no meetings).** Reply to every overnight reply FIRST (speed = trust). Then 50–60 fresh personalized touches from the list. Personalize line 1 every time.

**11:00–14:00 — CONVERT.** Run booked demos/setup calls. Send snapshots to repliers. Close verbal yeses → payment link on the spot. White-glove setup calls for new founding customers.

**14:00–16:00 — FOLLOW-UP + CONTENT.** Fire scheduled follow-ups (cadence in `02`). Post 1 LinkedIn update (progress, a finding, "X of 20 spots left"). Engage comments/DMs from the launch post.

**16:00–17:00 — ADD + PLAN.** Add 50 fresh prospects. Update the tracker (every field). Book tomorrow's demos. Tee up tomorrow's first 20 touches so you start fast.

**Evening — protect sleep.** A foggy founder closes nothing. This is a 15-day sprint, not a lifestyle. Recovery is part of the plan.

## Weekly management gates
- **Day 7 review:** If closes < 6 → the *message or targeting* is wrong, not the effort. Fix the offer framing / segment before adding volume. If replies high but closes low → the demo leaks; tighten the call (lead with the report on screen, ask for the close).
- **Day 10 checkpoint:** If behind, run the contingency in `01` (drop to monthly-first, widen ICP, add the €290 one-time report SKU).

## The 5 numbers on the wall (nothing else matters)
**Touches · Snapshots · Demos · Closes · Spots left.** Update daily in the tracker Dashboard. If touches are short, nothing downstream can hit. Outbound volume is the one input you fully control — protect the morning block above all.

## Close mechanics (turn demo → paid)
1. Lead the call with their **snapshot result + the report on screen** (proof, not pitch).
2. Name the offer once, clearly: "€79/mo locked for life, I set it up now, free to cancel, 1 of {N} founding spots."
3. **Ask for the close directly:** "Want me to set you up as a founding customer today?" Then silence.
4. On yes → payment link in chat → book the 30-min white-glove setup.
5. On "let me think" → "totally — the spots are filling and the price locks only for founders; can I hold one for you for 48h?" (honest soft scarcity).

## Onboarding a new founding customer (same day)
30-min call: integrate the one line together → run their baseline calibration → generate their first readiness report live → give the badge → add them to the founder Slack → ask (day 3) for one intro. First value moment within hours = retention + referral.

## Honesty guardrails (we're a compliance brand — these win deals, not lose them)
- "Audit-ready / evidence," never "compliant / certified."
- Badge = "Monitoring Active" (true), never "Compliant" (verdict).
- Real scarcity only — if 8 spots are left, say 8.
- Honor every opt-out instantly + forever.
- Aug 2 2026 is the date; note the possible Omnibus delay if asked — urgency is real regardless (procurement asks now).

## After Day 15

- Turn the 20 founding customers into: testimonials, case studies, reference calls, the trust-badge backlinks, and 1 intro each → fuel for the standard-pricing motion in `gtm/02-sales` + `gtm/05-marketing-launch`.
- Founding 20 full? Open Wave 2 at €89, or switch to public pricing + waitlist. The deadline keeps selling until Aug 2026.
