## Demo data for screenshots / live demos
The repo has `seed_demo.py` — run it to populate realistic drift/probe data so the dashboard and the generated PDF look real in demos. Never demo against an empty database.

## Asset usage rules (claims guardrails apply)
- The deck/one-pager say "compliance *evidence*," never "makes you compliant." Keep it that way.
- Add real design-partner logos to the deck (slide 1 strip / a traction slide) once you have consent.
- Date any competitor comparison; pricing shown is competitors' as of 2026-05 — refresh before sending.
- Swap `[placeholders]` (address, entity) once the legal entity is registered.
